PROJECT, CRM, DOCUMENT and more

A must tool for small or mid size of any distributed team

CRM
Build your customer database, manage business processes, analyze potential deals success and track sales with all the essential features of CRM software: Accounts, Contacts, Deal Opportunities, Tasks and Communication history.
Project Management
Keep latest updates of your projects includes milestones, tasks and task groups. You can also send message to team to announce an important event or see reports give you the correct understanding of your project performance
Document Management
Work with files in your own MyCollab document storage and Dropbox accounts. You are able to do any access operations such as copy, delete, move, name and upload files in all storage providers.
Online Document Editor
Work with files in your own MyCollab document storage and Dropbox accounts. You are able to do any access operations such as copy, delete, move, name and upload files in all storage providers.

Issues Management
Manage project issues could not be easier. Charts, email notification, time logging, reports will help the development team keep on track all of the aspects of project quality.  

Time Tracking
Log the effort you spend on assignments of your project. Show effort distribution in week, export them to attach to invoice and send to your customer. You can see effort distribution of your team and every project member.

Activity stream and audit logging
All users’s activities are logged into the system, and MyCollab allows users to view the recent activities with the detail description  and comprehensive history data as well.  

Users & Roles Management
Users are able to add as many users, roles as possible. Users can custom the permissions of roles and make them affected right away. Users also can change the profile information, change the avatar or custom the MyCollab theme as well.

MyCollab News

Mar, 12th, 2015: MyCollab 5.0.2 released
MyCollab 5.0.2 is a minor release that fixed several minor bugs reported by the customers, they are listed in the release notes. This release is compatible with the previous MyCollab releases.

Mar, 4th, 2015: MyCollab 5.0.1 released
MyCollab 5.0.1 is a feature release that includes a number of new features and bug fixes. We have improved the UI features, improve the screen navigator workflow and especially the application performance is much improved when the site is heavy load.

Feb, 17th, 2015: MyCollab 5.0.0 released
We have improved the UI design of MyCollab, added more features base on users’ feedbacks also simplified several major workflow of existing functionalities. The site performance is improved to make the loading process is faster, and all libraries are up-to-date.

Dec 1st, 2014: MyCollab 4.5.5 released
This minor release fixes the critical issue of threas-safe in saving bugs and tasks in multi thread accessing. In addition, we upgrade several important libraries up to date.

Nov 21st, 2014: MyCollab 4.5.4 released
We improve the stability of MyCollab mobile edition, and make the localization of any language show the friendly messages.

Nov 1st, 2014: MyCollab 4.5.3 released
The new mobile module is included in this release. In addition, MyCollab generates thumbnails for every image attachment which helps user can preview attachment easily. Several bug fixes are addressed and fixed also.

Oct 20th, 2014: MyCollab 4.5.2 released
This release focuses to the stability, ease of installation, upgrade third-party libraries versions, and compact the size of distribution.

Sep 23th, 2014: MyCollab 4.5.1 released
This release is an incremental feature release based on and compatible with earlier stable MyCollab releases.

Sep 5th, 2014: MyCollab 4.5.0 released
MyCollab development team is proud to announce the availability of MyCollab 4.5.0. This release mostly focuses into internal API improvements, minor bug fixes and other various UI polishes and improvements.
